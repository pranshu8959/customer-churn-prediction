# Customer Churn Prediction

This project predicts whether a customer will churn using machine learning algorithms like Random Forest.

## Features
- EDA and data cleaning
- Feature selection and encoding
- ML model training and evaluation
- Model deployment ready

## How to Run
1. Open `churn_predictor.ipynb` in Jupyter or Colab
2. Install requirements using `pip install -r requirements.txt`
3. Run all cells to train the model and view results
